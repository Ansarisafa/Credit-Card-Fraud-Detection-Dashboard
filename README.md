# Credit Card Fraud Detection Dashboard

## Overview

This project focuses on analyzing credit card transaction data to detect fraudulent activity. A professional interactive dashboard was developed using Power BI to visualize fraud distribution, transaction patterns, and key performance indicators.

## Objectives

* Analyze credit card transaction data
* Identify fraudulent vs normal transactions
* Visualize fraud distribution
* Compare transaction behavior
* Generate actionable insights

## Tools & Technologies

* Power BI
* SQL (SQLite)
* Data Visualization
* Data Cleaning
* DAX (Power BI)

## Dataset Features

* Transaction Amount
* Fraud Status (0 = Normal, 1 = Fraud)
* Multiple anonymized transaction attributes
* https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud?utm_source=chatgpt.com
## Dashboard KPIs

* Total Transactions
* Fraud Percentage
* Fraud Transactions Count
* Average Transaction Amount

## Visualizations

* Donut Chart – Fraud Distribution
* Bar Chart – Average Amount by Fraud Status
* Funnel Chart – Fraud vs Normal Transactions
* Distribution Chart – Transaction Amount Spread
* KPI Cards – Key Metrics

## Key Insights

* Fraud transactions represent less than 1% of total data
* Dataset is highly imbalanced
* Fraud transactions show different amount behavior
* Majority of transactions are normal

## Project Workflow

1. Data Loading in SQLite
2. Data Cleaning and Type Conversion
3. Duplicate Removal
4. Data Export to Power BI
5. Data Modeling
6. DAX Measures Creation
7. Dashboard Development
8. Insight Generation
